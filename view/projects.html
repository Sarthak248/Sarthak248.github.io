Delhi Metro - Shortest Route Finder

The Delhi Metro Shortest Route Finder is a web-based application built using JavaScript and React, designed to help commuters efficiently navigate the complex Delhi Metro network with its intricate interchanges. The application utilizes Dijkstra's algorithm, a widely-used graph traversal algorithm, to find the shortest route between two stations, ensuring optimal travel time and convenience.

Key Features:

  • User Friendly Interface

    The application boasts an intuitive and user-friendly interface, allowing commuters to easily input their desired source and destination stations. The interface is designed to accommodate even those less familiar with the Delhi Metro's extensive network.

  • Alternate Routes and Interchange Information

    In instances where multiple routes are available, the application presents alternate options to users. Additionally, it provides essential information about complex interchanges, helping commuters navigate these areas seamlessly.

Web Scraping Prototype - Zivame.com

The Automated Web Scraping Prototype is a Python-based solution that streamlines the process of fetching and exporting description details from various websites. Leveraging powerful frameworks such as Selenium, Pandas, Beautiful Soup, and Webdriver, the project automates data extraction and conversion to a CSV format, significantly reducing manual effort by almost 90%.

  • Efficient Web Scraping

    The prototype employs Selenium and Webdriver to simulate a web browser, enabling dynamic loading of pages with JavaScript-rendered content. This ensures that even websites utilizing AJAX and client-side rendering can be efficiently scraped.

  • Client-Defined Specifications

    The prototype allows clients to define their specific data requirements, such as target websites, search parameters, and the desired details to be extracted. This flexibility ensures that the solution can be tailored to meet individual needs.

  • Scalability and Modularity

    The project is designed to be scalable and modular, capable of handling multiple sources and websites simultaneously. Clients can easily extend the solution to include additional websites or data fields as needed.

Sudoku Solver

The Sudoku Solver and Automation project is an impressive Python-based application that tackles Sudoku puzzles of any difficulty level with remarkable efficiency. Powered by the Backtracking algorithm, the solver skillfully navigates through the intricacies of the puzzle to find the correct solution.

  • Selenium Automation

    The project leverages Selenium, a popular web automation tool, to interact with the website https://grid.websudoku.com/?level=4. Selenium automates the Sudoku puzzle generation process by scraping a problem from the website, feeding it to the solver, and subsequently displaying the solution on the website.

  • Real-Time Output

    Upon successful completion of the puzzle-solving process, the solution is instantly displayed on the website, giving users immediate feedback on their queries. This real-time output enhances the user experience and promotes efficient puzzle-solving.
