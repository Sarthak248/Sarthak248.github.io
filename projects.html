Enterprise Management System

A spring boot project that serves as a robust Enterprise Management System, providing RESTful APIs to perform CRUD operations.

Built with ORM tools like Hibernate and JPA, used JWT and Auth 2.0 for authentication, JDBC connection with MySQL.

The project thoroughly uses the Controller, Service, DAO and DTO architecture.

Includes functionalities like Security and Authentication, Data Validation, Exception Handling, Custom Responses for API clients like Postman and Browsers, ManyToOne and OneToMany mapping of entities, Pagination and Sorting and more...
